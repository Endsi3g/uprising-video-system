# Uprising Video System

A minimalist video automation system designed to streamline video production workflows.

## Description

This project provides a comprehensive solution for managing and automating video content creation. It includes features for content ingestion, script generation, and performance analysis, offering a clean and efficient workspace.

For internal documentation and information on how to configure or run the project, please refer to the `docs/` directory.

## Technical Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
