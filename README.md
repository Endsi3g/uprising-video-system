# Uprising Video System

A minimalist video automation system designed to streamline video production workflows.

## Description

This project provides a comprehensive solution for managing and automating video content creation. It includes features for content ingestion, script generation, and performance analysis, offering a clean and efficient workspace.

## Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   Copy `.env.example` to `.env.local` and configure the required values.

3. Start the development server:
   ```bash
   npm run dev
   ```

## Usage

Access the application through the browser. The system provides a dashboard for overseeing video projects, a library for asset management, and tools for script generation.

## Technical Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase
